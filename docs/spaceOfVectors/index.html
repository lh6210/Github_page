NOTES:
  • A real vector space is a set of vectors together with rules for vector addition and for multiplication by real numbers.
  • The space $ \mathcal{R}^n$ consists of all column vectors $ \mathbf{v}$ with $n$ components (real numbers); A vector whose $n$ components are complex numbers lies in the space $ \mathcal{C}^n$.
  • The space $ \mathcal{M}$ consists of all real $2$ by $2$ matrices.
  • The space $ \mathcal{F}$ consists of all real functions $f(x)$.
  • The function space $\mathcal{F}$ is infinite-dimensional. A smaller function space is $\mathcal{P}$, or $\mathcal{P}_n$, containing all polynomials $a_0 + a_1x + \cdots + a_nx^n$ of degree $n$.
  • The space $\mathcal{Z}$ consists only of a zero vector.
  • A subspace of a vector space is a set of vectors (including $0$) that satisfies two requirements: If $ \mathbf{v}$ and $ \mathbf{w}$ are vectors in the subspace and c is any scalar, then
    • $ \mathbf{v} + \mathbf{w}$ is in the subspace
    • $c \mathbf{v}$ is in the subspace
  • A subspace containing $ \mathbf{v}$ and $\mathbf{w}$ must contain all linear combinations $c \mathbf{v} + d \mathbf{w}$.
  • Inside the vector space $ \mathcal{M}$ of all $2$ by $2$ matrices, here are two subspaces:
    • $\mathcal{U}$: all upper triangular matrices $\left( \begin{matrix} a \; b \\ 0 \; d \end{matrix} \right)$
    • $\mathcal{D}$: all diagonal matrices $\left( \begin{matrix} a \; 0 \\ 0 \; d \end{matrix} \right)$
  • The column space of $A$, $C(A)$ consists of all linear combinations of the columns of matrix $A$. The columns span the column space.
  • The system $Ax = b$ is solvable if and only if $b$ is in the column space of $A$.
  • Instead of columns in $\mathcal{R}^m$, we could start with any set $\mathcal{S}$ of vectors in a vector space $\mathcal{V}$. To get a subspace $\mathcal{SS}$ of $\mathcal{V}$, we take all combinations of the vectors in that set:
    $\mathcal{S}$ = set of vectors in $\mathcal{V}$ (probably not a subspace)
    $\mathcal{SS}$ = all combinations of vectors in $\mathcal{S}$ (definitely a subspace)
    $\mathcal{SS} = $ all $ c_1 \mathbf{v}_1 + \cdots + c_N \mathbf{v}_N =$ the subspace of $\mathcal{V}$ spanned by $\mathcal{S}$
    Always $\mathcal{SS}$ is the smallest subspace containing $\mathcal{S}$. The columns "span" the column space.
